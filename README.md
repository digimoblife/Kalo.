## ğŸš€ Cara Instalasi (Local Development)

Ikuti langkah ini untuk menjalankan aplikasi di komputer lokal Anda:

### 1. Prerequisites
Pastikan Anda sudah menginstall:
* Flutter SDK (Versi terbaru)
* Git

### 2. Clone Repository
git clone [https://github.com/USERNAME_GITHUB_KAMU/kalo_app.git](https://github.com/USERNAME_GITHUB_KAMU/kalo_app.git)
cd kalo_app

### 3. Install Dependencies
flutter pub get

### 4. Konfigurasi Environment Variable (.env)
Buat file bernama .env di root folder project. Copy konfigurasi berikut dan isi dengan kredensial Supabase Anda:
SUPABASE_URL=[https://your-project-id.supabase.co](https://your-project-id.supabase.co)
SUPABASE_ANON_KEY=your-anon-key-here

### 5. Setup Database (Supabase)
Masuk ke SQL Editor di Dashboard Supabase Anda, dan jalankan perintah berikut untuk membuat tabel yang dibutuhkan:
-- 1. Table Profiles (Menyimpan data fisik user)
create table profiles (
  id uuid references auth.users not null primary key,
  email text,
  current_weight numeric,
  height numeric,
  birth_date date,
  gender text,
  activity_level text,
  daily_calorie_target int,
  current_streak int default 0,
  updated_at timestamp with time zone
);

-- 2. Table Foods (Database Makanan)
create table foods (
  id bigint generated by default as identity primary key,
  name text not null,
  calories int not null,
  protein numeric default 0,
  carbs numeric default 0,
  fats numeric default 0,
  serving_size numeric default 100,
  serving_unit text default 'gram',
  barcode text,
  is_external boolean default false,
  created_by uuid references auth.users,
  created_at timestamp with time zone default timezone('utc'::text, now())
);

-- 3. Table Food Logs (Jurnal Harian User)
create table food_logs (
  id bigint generated by default as identity primary key,
  user_id uuid references auth.users not null,
  food_id bigint references foods(id),
  log_date date not null,
  meal_type text, -- Breakfast, Lunch, Dinner, Snack
  portion numeric default 1.0,
  total_calories int,
  created_at timestamp with time zone default timezone('utc'::text, now())
);

-- 4. Trigger Auto-Create Profile (Opsional tapi Recommended)
-- Pastikan tabel profiles otomatis dibuat saat user signup
create or replace function public.handle_new_user() 
returns trigger as $$
